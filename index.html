2-1 Main Menu Animation

Catching mouse over event on anchor link and animate parent's adjecent element (in this case li) to move right and animate with easeOutBack effect. See the animation on the main menu at the top of the page.

$('#mainNav ul li a').hover(function(){ $(this).parent().next().stop(true, false).animate({ 'margin-left': '180px' }, 1000, 'easeOutBack'); }, function(){ $(this).parent().next().stop(true, false).animate({ 'margin-left': '0px' },1000); });
